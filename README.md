How to Run:
- Visit https://benjacobs-dev.github.io/PoolOfEchos_JS/

or:
- Download the code
- Open the `src/index.html` file (this is what the `RunMe.bat` does).

Controls (Click on the screen to access controls):
- WASD - Move Forward and side to side
- Shift - Move Up
- Ctrl - Move Down
- Mouse - Look Around
